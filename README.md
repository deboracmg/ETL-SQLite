## Descrição
Pipeline ETL em Python que transforma dados de vendas de um CSV e carrega métricas consolidadas por cliente em um banco SQLite.  

As métricas incluem:
- Quantidade de vendas por cliente
- Total gasto
- Ticket médio
- Classificação do cliente (Bronze, Prata ou Ouro)

---

## Tecnologias
- Python 3.13
- Pandas
- SQLite3
